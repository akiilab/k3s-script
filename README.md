# k3s-script

* **install.sh**: install k3s
* **uninstall.sh**: uninstall k3s
* **reset.sh**: clean up cluster data
* **server.sh**: install single master server
* **server-ha.sh**: install the first ha master server
* **server-ha-agent.sh**: install the second and more ha master server
* **worker.sh**: join to a master server
